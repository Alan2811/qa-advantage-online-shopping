# QA Advantage Online Shopping

Projeto de QA contendo testes manuais, bug reports e automação do fluxo de login do site Advantage Online Shopping.

## Estrutura do repositório
- **manual-tests/**: Contém casos de teste manuais e bug reports
  - `login-test-cases.md`
  - `login-bug-report.md`
  
## Tecnologias e ferramentas utilizadas
- QA Manual
- Cypress (para automação futura)
- Postman (para testes de API)
- Git / GitHub

## Objetivo
Demonstrar habilidades de QA aplicadas em cenário real de e-commerce:
- Criação de casos de teste
- Registro de bugs
- Planejamento de automação
