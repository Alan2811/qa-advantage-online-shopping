## Bug - Sistema permite login com campos vazios

**Ambiente:**  
- Plataforma: Web  
- Navegador: Chrome  

**Passos para reproduzir:**  
1. Acessar a página de login  
2. Deixar os campos de email e senha vazios  
3. Clicar no botão Login  

**Resultado atual:**  
O sistema permite o acesso do usuário

**Resultado esperado:**  
O sistema não deve permitir o acesso do usuário e deve exibir uma mensagem de erro informando que os campos são obrigatórios
